# Row Level Security Policies

Store Supabase Row Level Security policy notes and SQL files here.

Initial policy goals:

- Users can read/update their own profile.
- Users can read/update their own onboarding profile.
- Users can save/unsave their own properties.
- Business users can view assigned leads.
- Admin users can view/manage platform data.
- Public listing read access can be allowed when listings are active.
